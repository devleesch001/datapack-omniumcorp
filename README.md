# Omniumcorp Datapack

[![Release Datapack](https://github.com/devleesch001/datapack-omniumcorp/workflows/Release%20Datapack/badge.svg)](https://github.com/devleesch001/datapack-omniumcorp/actions)
[![License: AGPL v3](https://img.shields.io/badge/License-AGPL%20v3-blue.svg)](https://www.gnu.org/licenses/agpl-3.0)

Bienvenue dans le **Omniumcorp Datapack**, un ensemble de modifications et d'ajouts pour améliorer votre expérience de jeu sur Minecraft. Ce datapack se concentre sur l'ajout de nouvelles recettes de craft, de cuisson et une meilleure gestion des blocs de construction.

## Fonctionnalités

Le datapack apporte plusieurs modifications utiles au jeu de base :

### Nouvelles Recettes de Fabrication
- **Blocs de Quartz** : Possibilité de fabriquer des blocs de quartz à partir de blocs "cobbled" (pierre, ardoise des abîmes, granite, etc.) et de quartz.
- **Grès (Sandstone)** : Recettes simplifiées pour le grès et le grès rouge, incluant les variantes lisses (smooth).
- **Crying Obsidian** : Recettes personnalisées pour obtenir de l'obsidienne pleureuse.
- **Conversion Blé/Graines** : Utilitaires pour transformer le blé en graines.

### Améliorations de la Cuisine
- **Pain de Blé** : De nouvelles façons de cuire votre blé pour obtenir du pain via :
  - Le four classique (Smelting)
  - Le fumoir (Smoking)
  - Le feu de camp (Campfire Cooking)

### Système de Tags
Le datapack utilise des tags personnalisés (`#omniumcorp:cobbled_block`) pour regrouper de nombreux types de blocs rocheux (Stone, Andesite, Diorite, Granite, Deepslate, Blackstone, End Stone, etc.), permettant une plus grande flexibilité dans les recettes de craft.

## Installation

Téléchargez la dernière version du datapack depuis l'onglet [Releases](https://github.com/devleesch001/datapack-omniumcorp/releases).

## Développement

Le projet utilise GitHub Actions pour automatiser la création des archives et les releases :
- **Zip Datapack** : Crée automatiquement une archive du dossier `datapack`.
- **Release Datapack** : Gère le déploiement des nouvelles versions.
